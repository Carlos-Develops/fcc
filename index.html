HTML and CSS: Part 0


PARAGRAPHS:

Paragraphs are part of the basic HTML tags. They are used to describe an incoming paragraph to the browser. It's important to note that text can be emphasized in key areas.


LISTS:

Ordered Lists:

  1. First.
  2. Second.
  3. Third.

Unordered Lists:

  • First.
  • Second.
  • Third.

COMMENTS:

Comments are helpful to ignore HTML processing by the browser. It can be done by using the form:

<!-- HTML goes here -->

This prevents HTML code execution.
